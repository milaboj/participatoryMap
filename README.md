# participatory accessibility map
No AI tools were used in this assignment


This map allows users to report features that hinder accessibility like extreme hills, potholes, or cracked sidewalks in Seattle. This map is looking to help people who live in Seattle who may have a harder reason getting around due to a disability such as people in wheelchairs, older individuals, people on crutches, etc.

The web map can be found [here](https://milaboj.github.io/participatoryMap/docs/index.html).
